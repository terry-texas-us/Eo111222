
Color definition files are formatted as follows:

<Colors>
PenId=RedActive,GreenActive,BlueActive,RedStatic,GreenStatic,BlueStatic
.	
.
</Colors>

	Active colors (colorPalette) are used for Work and Active layers and Opened tracings.
	Static colors (grayPalette)are used for Static layers and Mapped tracings
	
	Viewed tracings use colorPalette[0] and grayPalette[0]


	Pens can be defined for all PenId (0 to 255) for colorPalette and (0 to 15) for grayPalette.
